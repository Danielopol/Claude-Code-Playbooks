## Instructions

1. Start logging businesses you visit
2. Use the template for detailed reviews
3. Quick-rate places for at-a-glance reference
4. Build comparison notes for decisions
5. Tag for easy searching

## Commands

```
"Add new restaurant: [name]"
"Rate [business] [1-5] stars"
"What's my best option for [type]?"
"Show all [category] businesses"
"What did I think of [business]?"
"Compare [type] options"
"Find places tagged [tag]"
```

## Search & Filter

Find businesses by:
- Category
- Rating (4+ stars)
- Neighborhood
- Price range
- Tags
- "Would recommend"
- Last visited

## Mobile Workflow

When out:
1. Quick-note the business name
2. Snap a photo of the card/menu
3. Add full review later at home

```
"Quick note: tried Maria's Bakery, loved the croissants"
```

Expand later with full review template.
