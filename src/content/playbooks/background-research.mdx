## What This Does

Lets you delegate research tasks while focusing on creative or primary work. Claude researches topics, compiles findings with sources, and presents summaries without interrupting your flow.

## Prerequisites

- Claude Code installed
- Research topics defined
- Output format preference

## Setup Instructions

### Step 1: Download the Template

Download the CLAUDE.md template below and save it to your research folder.

### Step 2: Define Research Briefs

Create a template for research requests:

```markdown
# Research Request

Topic: [What to research]
Purpose: [Why you need this]
Depth: [Quick scan / Standard / Deep dive]
Format: [Summary / Detailed / Annotated]
```

### Step 3: Start Researching

Kick off background research:

```
Research [topic] while I work on [other thing]
```

## Example Usage

```
"Find data on [specific question]"
"What are expert opinions on [topic]?"
"Fact-check this claim: [claim]"
"Compile sources on [topic]"
"Summarize what you found"
```

## Research Types

- **Quick Scan**: 15-30 min, surface-level, key points
- **Standard**: 1-2 hours, comprehensive, multiple sources
- **Deep Dive**: 4+ hours, exhaustive, primary sources

## Best Practices

1. **Be specific** - Clear questions get better answers
2. **Set scope** - Define depth and time investment
3. **Trust but verify** - Check important sources yourself
4. **Build on findings** - Use research as a starting point
