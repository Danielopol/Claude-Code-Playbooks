## What This Does

Helps plan events by coordinating calendars, researching venue options, comparing prices, and making recommendations. Works for private parties, team dinners, and entertainment bookings.

## Prerequisites

- Claude Code installed
- Event requirements defined
- Calendar access (or participant availability)

## Setup Instructions

### Step 1: Download the Template

Download the CLAUDE.md template below and save it to your events folder.

### Step 2: Define Your Event

Prepare event details:

```markdown
# Event Brief

Type: [Party, dinner, meeting]
Date range: [Flexible dates]
Group size: [Number]
Budget: [Total or per-person]
Location: [City/area]
```

### Step 3: Start Planning

Begin the planning process:

```
Plan a dinner for 20 people in [city] for [date range]
```

## Example Usage

```
"Find venues available [dates]"
"Compare these venue options"
"What's my best option under $[budget]?"
"Check calendar conflicts"
"Create a booking summary"
```

## What Gets Done

- Calendar coordination
- Venue research and comparison
- Price analysis
- Availability checking
- Recommendation with reasoning

## Best Practices

1. **Start with dates** - Lock attendance first
2. **Set budget early** - Narrows options quickly
3. **Have backups** - Things fall through
4. **Book ahead** - Good venues fill up
