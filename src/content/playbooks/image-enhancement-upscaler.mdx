## Example Commands

### Basic Improvements
```bash
# Sharpen an image
convert input.jpg -sharpen 0x1.0 output.jpg

# Enhance contrast
convert input.jpg -contrast-stretch 2%x1% output.jpg

# Auto-enhance (levels + contrast)
convert input.jpg -auto-level -normalize output.jpg

# Reduce noise
convert input.jpg -enhance -enhance output.jpg
```

### Upscaling
```bash
# 2x upscale with Lanczos
convert input.jpg -filter Lanczos -resize 200% output.jpg

# Upscale to specific dimensions
convert input.jpg -filter Lanczos -resize 1920x1080^ output.jpg

# Upscale maintaining aspect ratio
convert input.jpg -filter Lanczos -resize 1920x1080 output.jpg
```

### Screenshot Enhancement
```bash
# Sharpen screenshot for clarity
convert screenshot.png -sharpen 0x0.5 -quality 95 enhanced.png

# Remove compression artifacts
convert screenshot.jpg -enhance -quality 95 cleaned.png

# Increase DPI for printing
convert screenshot.png -density 300 -units PixelsPerInch print-ready.png
```

### Batch Processing
```bash
# Enhance all JPGs in folder
for f in *.jpg; do
  convert "$f" -auto-level -sharpen 0x0.5 "enhanced_$f"
done

# Resize all images to 50%
mogrify -resize 50% *.jpg

# Convert PNG to JPG with quality
mogrify -format jpg -quality 90 *.png
```

## Quality Settings

### JPEG Quality
| Quality | File Size | Use Case |
|---------|-----------|----------|
| 95-100 | Large | Archival |
| 85-94 | Medium | Web high-quality |
| 70-84 | Small | Web standard |
| < 70 | Tiny | Thumbnails |

### PNG Optimization
```bash
# Optimize PNG without quality loss
convert input.png -strip -quality 95 output.png

# Reduce colors for smaller size
convert input.png -colors 256 output.png
```

## Tips

- **Start with source quality**: Enhancement can't create detail that doesn't exist
- **Preserve originals**: Keep untouched copies
- **Match the use case**: Web needs smaller files, print needs higher DPI
- **Don't over-sharpen**: Halos around edges = too much
- **Use PNG for screenshots**: Better quality than JPG for text/UI
