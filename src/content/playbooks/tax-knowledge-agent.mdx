## What This Does

Organizes tax documents into a searchable knowledge base. Answer questions about your income, deductions, and documentation with citations to source documents.

## Important Disclaimer

**This is an organizational tool, not tax advice.** Consult a qualified tax professional (CPA, EA) for tax decisions.

## Prerequisites

- Claude Code installed
- Tax documents (W-2s, 1099s, receipts)
- Folder structure for organization

## Setup Instructions

### Step 1: Download the Template

Download the CLAUDE.md template below and save it to your tax folder.

### Step 2: Organize Documents

Create a structured folder:

```
tax-2024/
├── CLAUDE.md
├── income/
│   ├── w2-employer.pdf
│   └── 1099-freelance.pdf
├── deductions/
│   ├── charitable/
│   └── business/
└── prior-returns/
```

### Step 3: Build Your Knowledge Base

Index your documents:

```
Index my tax documents for 2024
```

## Example Usage

```
"What income did I receive from [source]?"
"Show all my charitable donations"
"What documents am I missing?"
"Create a prep checklist"
"When did I receive [document]?"
```

## What Gets Organized

- Income documents (W-2, 1099 variants)
- Deduction receipts
- Credit documentation
- Prior year returns
- Key dates and deadlines

## Best Practices

1. **Organize early** - Don't wait until April
2. **Keep everything** - When in doubt, save it
3. **Document donations** - Get receipts immediately
4. **Consult professionals** - For complex situations
